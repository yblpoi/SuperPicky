SuperPicky - Installation Guide
================================

1. INSTALL THE APP
   Drag "SuperPicky.app" into the "Applications" shortcut in this window.
   Then launch it from your Applications folder or Launchpad.

   On first launch, macOS may show a security prompt.
   Go to System Settings → Privacy & Security → click "Open Anyway".

2. INSTALL THE LIGHTROOM CLASSIC PLUGIN (optional)
   SuperBirdIDPlugin.lrplugin adds bird ID results directly into Lightroom.

   Option A — Plug-in Manager (recommended):
     a. In Lightroom Classic: File → Plug-in Manager → Add
     b. Navigate to your Applications folder and select SuperBirdIDPlugin.lrplugin
        (or copy it first to ~/Library/Application Support/Adobe/Lightroom/Modules/)
     c. Click "Add Plug-in" → Done

   Option B — Manual install:
     Copy "SuperBirdIDPlugin.lrplugin" to:
     ~/Library/Application Support/Adobe/Lightroom/Modules/
     Then restart Lightroom Classic.

3. GETTING STARTED
   Open SuperPicky, select a folder of photos, adjust settings, and click Start.
   Results are written back as EXIF/XMP metadata and can be filtered in Lightroom.
